Равномерная сходимость несобственных интегралов

Несобственный интеграл называется равномерно сходящимся по параметру \(\displaystyle \gamma \) на \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось)., если он сходится на \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось). и выполняется одно из следующих эквивалентных условий:

1) \(\displaystyle \forall \varepsilon >0\) \(\displaystyle \exists {{\eta }_{\varepsilon }}\ge a\) \(\displaystyle \forall \eta \in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\) \(\displaystyle \forall \gamma \in\) \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось). \(\displaystyle \left| \int\limits_{\eta }^{\mathbf{B}}{f(x,\gamma )dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\),

2) \(\displaystyle \forall \varepsilon >0\) \(\displaystyle \exists {{\eta }_{\varepsilon }}\ge a\) \(\displaystyle \forall \eta \in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\) \(\displaystyle \underset{\gamma \in \Gamma }{\mathop{\sup }}\,\) \(\displaystyle \left| \int\limits_{\eta }^{\mathbf{B}}{f(x,\gamma ) dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\),

3) \(\displaystyle \underset{\eta \to \mathbf{B}}{\mathop{\lim }}\,\underset{\gamma \in \Gamma } {\mathop{\sup }}\,\left| \int\limits_{\eta }^{\mathbf{B}}{f\left( x,\gamma \right)dx} \right|=0.\)

Теорема 14 (критерий равномерной сходимости Коши). Для равномерной по параметру \(\displaystyle \gamma \) на \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось). сходимости несобственного интеграла

\(\displaystyle I(\gamma )=\int\limits_{a}^{\mathbf{B}}{f(x,\gamma )dx}\), \(\displaystyle \gamma \in\) \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось).

необходимо и достаточно, чтобы

\(\displaystyle \forall \varepsilon >0\) \(\displaystyle \exists {{\eta }_{\varepsilon }}\ge a\) \(\displaystyle \forall {\eta }'\in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\,\,\,\forall {\eta }''\in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\) \(\displaystyle \underset{\gamma \in \Gamma }{\mathop{\sup }}\,\)\(\displaystyle \left| \int\limits_{{{\eta }'}}^{{{\eta }''}} {f(x,\gamma )\,dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\).

Следствие (связь с функциональными рядами). Равномерная по \(\displaystyle \gamma \) сходимость интеграла \(\displaystyle I(\gamma )\), \(\displaystyle \gamma \in\) \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось). эквивалентна равномерной сходимости всех рядов вида \(\displaystyle \sum\limits_{n=0}^{+\infty }{\int\limits_{{{c}_{n}}}^{{{c}_{n+1}}} {f(x,\gamma )dx}}\), где \(\displaystyle \left\{ {{c}_{n}} \right\}\) – произвольная, сходящаяся к \(\displaystyle \mathbf{B}\) последовательность.
